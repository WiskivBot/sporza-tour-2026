## 💡 Strategie

### Opbouw
- **GC-dekking**: Pogačar + Vingegaard + Evenepoel (~2500 pt uit bergritten + GC)
- **Sprint**: Philipsen + Pedersen + De Lie + Girmay (~2000 pt uit vlakke ritten)
- **Allround**: Van der Poel + Pidcock + Godon (ritwinsten uit alle zones)
- **Toekomst**: Seixas + Del Toro (witte trui + jongerenpunten)
- **Bus**: Oldani + Howson + Van Lerberghe (€2M elk — budgetbuffer)

### Transferstrategie
- **3 gratis transfers** — gebruik ze strategisch
- Week 1: Hou sprinters in de basis
- Week 2 (Alpen): Switch naar klimmers
- Week 3 (Alpe d'Huez dubbel!): Extra klimmers erbij
- **Hou €6-10M over** voor mid-season noodtransfers

### Kopman-advies
**Kies Pogačar** — hij levert veruit de meeste punten. Alternatief: Vingegaard als underdog.
